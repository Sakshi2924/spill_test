Admin Not Here

Admin

The admin console runs on a separate server

This static deployment serves the public site only. The admin dashboard — with image uploads, content editing, and authentication — needs a long-running Node process with persistent storage, which static hosting can't provide.

To access the admin

Run it locally with npm start, or deploy the repo to a Node-capable host such as Railway, Fly.io, Render, or a VM. See DEPLOYMENT.md in the repository for a walkthrough.
